## Noxaudit v1.2.3 Release

We have just shipped v1.2.3 of Noxaudit. This release addresses critical feedback regarding output truncation during large-scale security scans.

### Key Improvements
- **Expanded Completion Buffer (PR #148):** We have increased the `max_completion_tokens` from 4,096 to 16,384. This change by @jeff-atriumn ensures that complex audits no longer hit the 4k token ceiling, which previously caused truncated JSON payloads or incomplete vulnerability reports.
- **Stability Enhancements:** This version builds on the 1.2.2 release (PR #145) to ensure smoother handling of long-form audit responses.

### Upgrading
To pull the latest changes, update your global or project-level dependency. No changes to your CLI invocation logic are required to benefit from the expanded window.
