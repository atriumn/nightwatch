## Noxaudit v1.1.8 is now live

Following the successful merge of PR #131 and the v1.1.7 release, we are shipping **v1.1.8** to further refine the auditing experience for your local codebases.

### Key Updates
- **Release Automation:** Integration of the latest chore(main) workflow from @jeff-atriumn, ensuring more consistent delivery cycles.
- **CLI Performance:** Under-the-hood improvements to the AST walker to handle deeper nesting patterns in large-scale projects.
- **Stability Fixes:** Resolved edge cases identified in v1.1.7 regarding environment variable parsing during audits.

### Migration
To upgrade your local CLI, run:
`npm install -g noxaudit@1.1.8` 

If you are using Noxaudit in your CI/CD pipelines, ensure your configuration points to the latest tag to benefit from these stability improvements.
