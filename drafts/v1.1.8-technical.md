## Noxaudit v1.1.8

We have just released v1.1.8, extending our scanning capabilities to the Dart ecosystem and streamlining our release automation.

### New Features & PRs
- **Full Dart Support (#133)**: We’ve added comprehensive support for Dart across all audit focus areas. This enables `noxaudit` to parse pubspec files and analyze Dart-specific security patterns.
- **Automated Release Pipeline (#131)**: We have formalized our release process to ensure more reliable versioning and artifact generation. This includes the groundwork laid in the v1.1.7 tag.

### Upgrading
Update your local installation to leverage the new Dart focus areas:

- Run your package manager's update command (e.g., `npm install -g @atriumn/noxaudit`).
- Verify the installation with `noxaudit --version`.
- Execute `noxaudit scan` to test the new Dart integration on your Flutter or server-side Dart projects.

For a detailed breakdown of the internal changes, check the [v1.1.7 release notes](https://github.com/atriumn/noxaudit/releases/tag/v1.1.7) and the subsequent 1.1.8 release tags.
