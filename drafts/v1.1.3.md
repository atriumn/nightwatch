Noxaudit v1.1.3 is now available.

This release focuses on refining our cost calculation logic to ensure more accurate auditing across varied timeframes.

### Improved Pricing Accuracy
We’ve updated `pricing.py` to remove hardcoded "daily" assumptions from the output (PR #109). Previously, the engine could misrepresent costs if the audited period wasn't strictly 24 hours. This fix ensures that reported figures reflect your actual data range, improving reliability for custom audit windows.

### Release Details
Following the v1.1.2 tag, this version consolidates these pricing fixes into a stable build. You can review the full changes and pull requests on [GitHub](https://github.com/atriumn/noxaudit/releases).
