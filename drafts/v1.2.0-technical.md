## Noxaudit v1.2.0 Release

Noxaudit v1.2.0 is now available. This release focuses on optimizing the initial scanning phase to reduce latency and improve the developer experience during local audits.

### Key Improvements

- **Synchronous Pre-pass API (PR #139):** We have migrated the pre-pass logic from the Batch API to a new Sync API. Previously, the CLI relied on batch processing which introduced polling overhead. By switching to a synchronous flow, the discovery phase is now significantly faster, providing more immediate feedback when you trigger a scan.
- **Stability Updates:** Following the v1.1.10 release, we've refined internal release workflows and dependency management to ensure a smoother installation process.

### Updating

To pull the latest changes, update your CLI version:

- `npm install -g @atriumn/noxaudit@latest` 
- Or update your `package.json` / `requirements.txt` to point to version `1.2.0`.

If you encounter any issues with the new sync API behavior in your CI environments, please open an issue in the repository.
