Hi there,

We’ve just released **Noxaudit v1.1.1**. This release follows the major v1.1.0 milestone, focusing on performance stabilization and ensuring the latest engine improvements are production-ready.

### What’s New
This update acts as the final polish for the architectural changes introduced in v1.1.0 (via PR #101). Our main objective was to streamline the audit execution path and resolve edge cases found during the v1.1.0 rollout.

### Impactful Changes
*   **v1.1.0 Stability:** Fully integrates the core auditing enhancements from our latest major release.
*   **Resource Efficiency:** Optimized internal data parsing to reduce memory overhead during large-scale audit runs.
*   **CI/CD Reliability:** Fixed minor regression bugs to ensure consistent reporting across distributed environments.

### Why Upgrade?
v1.1.1 ensures that the new features introduced in the 1.1.x series are robust enough for mission-critical workflows. If you’ve already moved to v1.1.0, this is a recommended patch to improve overall reliability.
