## Noxaudit v1.1.10 Shipped

This release introduces critical improvements to how the CLI handles large-scale codebases, specifically targeting context window limitations.

### Automatic Pre-pass for Large Files
The most impactful change in this version is the intelligent handling of token limits. When Noxaudit detects that your codebase or specific file set exceeds the model's context window, it now automatically enables the `pre-pass` mechanism. This prevents CLI failures and ensures comprehensive audits even on your most complex modules without requiring manual flag adjustments.

- **PR #137**: fix: auto-enable pre-pass when tokens exceed model context window (@jeff-atriumn)

### Release Lifecycle
Following the v1.1.9 chore updates, v1.1.10 stabilizes the release pipeline and ensures these context-aware fixes are available in the main distribution branch.

- **v1.1.9 Release**: [View Tag](https://github.com/atriumn/noxaudit/releases/tag/v1.1.9)
