## Noxaudit v1.2.2 is Live

Noxaudit v1.2.2 is now available for deployment. This release focuses on refining resource allocation and improving the efficiency of the pre-pass scanner.

### Performance & CLI Changes

- **Pre-pass Memory Optimization**: In PR #144, we reduced the pre-pass chunk headroom from 75% to 50%. This change decreases the memory overhead during initial file indexing, leading to faster scan initialization on large codebases without increasing the risk of buffer overflows.
- **Pipeline Stability**: Following the v1.2.1 tag (PR #142), we have stabilized the automated release workflow to ensure faster delivery of security definitions and engine updates.

### Upgrading

To update your local installation or CI environment, run:

- `npm install -g @atriumn/noxaudit@latest` 
- Or update your `package.json` to version `1.2.2`.

You can view the full diff and release notes on [GitHub](https://github.com/atriumn/noxaudit/releases/tag/v1.2.2).
